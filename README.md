# Libreta de Viajes

Starter Template for **N**etlify CMS, **E**leventy
## Live Demo

[https://helpful-dango-319c99.netlify.app/](https://helpful-dango-319c99.netlify.app/)

### Technologies used:

- [Netlify CMS](https://www.netlifycms.org/)
- [Eleventy](https://www.11ty.dev/)

## Getting Started

### 1\. Install dependencies

```
npm install
```

### 2\. Build the project to generate the first CSS

This step is only required the very first time.

```
npx @11ty/eleventy
```

### 3\. Run Eleventy

```
npm run start
```
